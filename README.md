# Sneakers Monsters™ Complete HTML Website

Static HTML/CSS/JS website rebuilt from the professional font version and expanded with fuller English/French content plus a complete bilingual blog section.

## Preview
Open `index.html` in a browser.

## Included
- English and French pages
- Blog index in English and French
- 4 English blog article pages
- 4 French blog article pages
- Reusable lead magnet sections on major pages
- Structured inquiry/contact forms with preformatted dropdown menus
- Budget range is now a dropdown
- Original font system preserved: Archivo Black, Barlow Condensed, Outfit
